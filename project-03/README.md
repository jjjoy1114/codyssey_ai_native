# 프로젝트 2: Google Forms 설문 응답 자동 저장 및 조건별 알림

## 과제 구분

- 과정명: AI Native Basic
- 과제명: 자유 주제 자동화 설계 및 구현
- 자동화 주제: 교육 프로그램 만족도 설문 자동 분류 및 알림
- 사용 예정 도구: Google Forms, Google Sheets, Make, Gmail

---

## 1. 프로젝트 개요

교육 프로그램 만족도 설문이 제출되면 응답 내용을
Google Sheets에 자동으로 저장한다.

만족도가 낮거나 상담을 요청한 응답이 접수되면
담당자에게 이메일 알림을 보내도록 자동화한다.

## 2. 자동화가 필요한 이유

기존에는 담당자가 Google Sheets를 직접 열어
새로운 설문 응답을 확인해야 했다.

이러한 방식은 상담 요청이나 불만족 응답을
늦게 발견할 수 있다는 문제가 있다.

따라서 응답을 자동으로 분류하고,
확인이 필요한 응답이 접수되면 담당자에게
즉시 알림을 보내는 자동화를 설계하였다.

## 3. 사용 도구

| 도구 | 사용 목적 |
|---|---|
| Google Forms | 만족도 및 상담 요청 수집 |
| Google Sheets | 설문 응답 저장 |
| Make | 조건 확인 및 자동화 실행 |
| Gmail | 담당자 이메일 알림 |

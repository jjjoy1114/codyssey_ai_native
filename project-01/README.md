# 🤖 디지털 전환 가이드 챗봇 '다알려드리미' 프로젝트

본 프로젝트는 마케팅 인력과 시간이 부족하고 디지털 환경에 익숙하지 않은 자영업자 및 소상공인을 지원하기 위해 기획된 LLM 기반 디지털 전환 가이드 챗봇 **'다알려드리미'**의 개발 및 검증 기록입니다.

복잡한 IT 전문 용어를 배제하고 일상적인 언어를 사용하여 사용자의 심리적 장벽을 낮추고, 단계별 가이드를 통해 소상공인이 직접 홍보 영상을 제작하고 SNS 마케팅을 실행할 수 있도록 돕습니다.

---

## 🎯 프로젝트 개요
* **챗봇명:** 다알려드리미
* **기반 모델:** ChatGPT (무료 버전)
* **타겟 사용자:** 40~60대 자영업자 및 소상공인
* **핵심 가치:** 전문 용어 배제, 공감형 어조를 통한 심리적 안정감 제공, 즉각 실행 가능한 단계별 가이드
* **주요 검증 항목:** 환각 방지(존재하지 않는 기능 거부), 멀티턴 문맥 유지(업종 기억), 심리적 안정(실수 안심시키기)

## 🛠️ 핵심 성공 요인 (Key Insights)
1. **구체성 > 모델 종류:** 프롬프트의 구체성을 높이는 것이 모델의 체급을 바꾸는 것보다 어조 관리 및 용어 순화에 더 큰 성능 개선을 가져옴을 입증했습니다.
2. **사전 설계를 통한 위험 예방:** 발생한 오류를 사후에 수정하는 대신, 금지어 목록 지정 및 우선순위 규칙(심리적 안정 > 정보의 정확성) 설정을 통해 챗봇의 이탈을 사전에 차단했습니다.
3. **무료 환경 재현성 확보:** 유료 환경에서 검증된 우수한 성능이 무료 환경(ChatGPT Free)에서도 100% 동일하게 재현되어 소상공인의 실사용 가능성을 확보했습니다.

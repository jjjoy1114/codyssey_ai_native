## **2\. 타겟 페르소나와 문제(Pain Point) 정의** 

* **타겟 페르소나 정의:** 영상 편집 및 디지털 협업 툴을 비즈니스에 도입하고자 하는 자영업자·소상공인  
  - 주변에서 실제 알게 된 사장님들 사연을 재구성  
  - "카페를 운영하는 40대 김미영 사장님은 하루 12시간 매장에 있어 편집할 시간이 없고, 스마트폰 앱 사용도 서툴러 인스타그램 릴스 제작 하고 싶었지만 인스타그램 홍보 포기했다"  
    * 이름/나이/직업 : 김미영/ 45세/ 동네 카페 사장  
    * 디지털역량 : 스마트폰 기본 사용 가능, 편집 앱은 어려움  
    * 현 재 행 동 : 사진만 가끔 인스타 업로드  
    * 목 표 : 카페 홍보 영상을 인스타그램에 주 2회 올리고 싶음  
    * 장 애 물 : 시간 부족, 편집 기술 없음, 외주 비용 부담  
* **문제 (Pain Point) :**   
  - 최신 디지털 도구(캡컷, 구글 워크스페이스, 클라우드 시스템 등)가 비즈니스 및 일상 생활에서 효율성을 대폭 높여주고 있음에도 불구하고, 디지털 초보자들은 복잡한 UI와 낯선 IT 전문 용어로 인해 초기 진입에 큰 어려움을 겪고 있음을 인지  
  - 기존 AI 검색이나 매뉴얼은 기술 중심적인 용어로 서술되어 있어 심리적 장벽을 가중함을 인지  
  - 주로 5인 이하의 사업체의 경우, 숏폼 영상 외주 비용 건당 수만\~수십만 원으로 경영 부담이 가중되고, 개인이 SNS 트렌드 파악이 어려움을 인지  
  - LLM을 이용하더라도 사전 학습된 모델에 예시 없이 지시만 내리는 제로 샷 Zero shot, 질문이나 한개의 예시 만 드는 원샷 one shot 질문으로 원하는 대답을 잘 얻지 못하는 경우가 많음을 인지  
      
* **해결방안 :**  
  - **입력 데이터 형태:** 자연어 기반의 모호하거나 일상적인 디지털 기기/소프트웨어 조작 관련 질문 텍스트.  
  - **출력 규격:** IT 전문 용어가 배제되고 일상적 비유로 풀이된 친절한 설명문, 숫자로 명시된 단계별 실행 가이드, 핵심 주의사항 및 무료/유료 여부 명시.  
  - **핵심 가치:** IT 전문 용어 배제 및 일상 언어 활용, 사용자 중심의 심리적 안정감 제공, 단계별 가이드를 통한 즉각적 실행 가능.  
  - **성공 지표:** 질의 의도 파악 정확도 90% 이상, 3턴 이내 문제 해결 비율 80% 이상, 사용자 만족도 4.5/5.0 이상.


## **3\. 해결 방안 (LLM 활용 포인트)**

*  LLM을 활용하여 자연어 기반의 모호하거나 일상적인 디지털 기기/소프트웨어 조작 부터 SNS 마케팅 트렌드에 맞는 시나리오 기획, 영상제작, 편집과정에 문제점이 생겼을 때 해결이 가능한 무료 챗봇 제작 배포 

## **4\. 챗봇의 페르소나 정의 및 매뉴얼**

* **이름 및 역할:** 인내심 많고 친절한 디지털 튜터 '***다알려드리미***'  
* **전문 분야:** SNS 마케팅, 모바일 동영상 편집, 스마트 비즈니스 도구 활용 가이드  
* **말투:** 항상 존댓말을 사용하며, 다정하고 칭찬을 아끼지 않는 격려의 어조. "어려우시죠? 저와 함께 천천히 따라 해보시면 금방 하실 수 있어요\!"와 같이 공감형 문구를 첫머리에 배치합니다.  
* **금지 사항:** '클립', '타임라인', '렌더링', '인코딩', '레이어' 등 전문 용어를 가공 없이 원어 그대로 사용하는 행위 금지. 사용자의 실수를 지적하거나 단정적인 거절 문구 사용 금지.  
* **우선순위:** 1순위 사용자의 심리적 안정감 및 친절함, 2순위 정확성, 3순위 정보의 양

### **5.2. 항목별 상세 분석**

① 공감형 문구 도입 효과

* 간단한 지시(A): "반갑습니다", "좋은 질문입니다" 수준의 일반적인 인사말에 그쳤습니다.  
* 구체적인 지시(B): 모든 답변의 첫머리에 "어려우시죠? 😊"와 같은 공감형 문구를 배치하여 지시사항이 100% 충실히 반영되었습니다.

② 전문 용어 회피 전략 

- 3번 질문("타임라인에서 클립 자르는 법")을 통해 모델별 대응 차이를 확인했습니다.

| 모델명 | 간단한 지시 (A) | 구체적인 지시 (B) |
| :---: | :---: | :---: |
| **GPT** | 타임라인 개념 직접 설명 | 용어 재정의 및 쉬운 표현 대체 |
| **Claude** | 전문 용어 그대로 노출 | 금지어 원천 차단 및 비유적 표현 사용 |

* 시사점: GPT는 용어를 정의한 후 사용하는 유연함을 보였고, Claude는 금지어를 완전히 회피하는 엄격한 전략을 취했습니다.

③ 환각(Hallucination) 거부 능력  
"세 손가락 무지개 자막"과 같은 허위 질문에 대해 4개 대화방 모두 기능을 부정하며 답변했습니다. 다만, 답변의 어조에서 차이가 발생했습니다.

* GPT(구체적 지시): "제가 아는 범위에서는 확인되지 않습니다"라며 겸손하고 정직한 태도 견지.  
* Claude(구체적 지시): "전혀 걱정 마세요, 그럴 수 있어요 😊"라며 친절함과 안정감을 우선순위로 배치.  
* 참고: Claude의 경우 친절함을 위해 답변을 과하게 완곡하게 표현한 경향이 있어, 향후 '정확성' 중심의 튜닝이 필요할 수 있습니다.

④ 맥락(업종) 유지 능력

* Claude: 사용자가 밝힌 업종(안전용품) 정보를 전 답변에 걸쳐 지속적으로 반영하는 능력이 탁월했습니다.  
* GPT: 업종 예시를 적절히 반영했으나,Claude에 비해 맥락 유지의 일관성이 다소 낮게 나타났습니다.

### **5.4. 종합 결론**

* 구체적 지시(B)의 개선 효과: 공감형 문구, 전문 용어 회피, 어조 관리 등 모든 항목에서 두 모델 모두 뚜렷한 성능 향상을 보였습니다. 명시되지 않은 지시사항(공감 문구 등)은 간단한 지시에서는 나타나지 않았다는 점이 핵심입니다.  
* 모델별 이행 차이: 동일한 지시에도 Claude는 금지어 완전 회피 및 맥락 유지에서 더 엄격하고 철저한 모습을 보였으며, GPT는 환각 거절 시 더 정직하고 겸손한 표현을 사용하는 특성을 보였습니다.  
* 최종 평가: 프롬프트의 구체성은 모델의 종류보다 더 큰 성능 개선 효과를 가져옵니다.

* #### **지시문(Instructions) 구성**

```
# ⚙️ 설정 변수 (재사용 시 이 블록만 수정하세요)
- [봇 이름]: 다알려드리미
- [전문 분야]: SNS 영상 편집
- [대상 사용자]: 디지털 도구가 익숙하지 않은 50~60대 소상공인
- [호칭]: 사장님
────────────────────────────
# 역할
너는 '[봇 이름]'이라는 이름의 [전문 분야] 튜터야.
[대상 사용자]가 어려움 없이 목표를 달성하도록 돕는 것이 너의 임무야.
# 말투 규칙
1. 모든 답변은 공감 문구로 시작한다. (예: "어려우시죠? 😊")
2. 사용자를 '[호칭]'이라고 부른다.
3. 설명은 반드시 '순서대로 번호를 매긴 단계'로 제시한다.
# 금지 사항
1. 전문 용어를 그대로 쓰지 않는다. (예: '타임라인' → "영상이 놓인 줄")
2. 단정적인 거절 표현이나 실수를 지적하지 않는다.
# 환각 방지
- 아는 척하며 지어내지 않는다. 대신 부드럽게 안내하고 안심시킨다.
# 우선순위 (규칙이 충돌할 때 이 순서를 따른다)
1순위: 사용자의 심리적 안정감과 친절함
2순위: 정확성
3순위: 정보의 양
→ 정확하지만 불친절한 답보다, 친절하면서 정확한 답을 지향한다.
# 사용자 맥락 활용
- 사용자가 자신의 상황(업종·목적 등)을 알려주면, 이후 예시에 반드시 반영한다.
  예: 안전용품점 → "안전모 신상품 소개 영상" 같은 예시로 설명
# 답변 형식
1. 공감 인사 한 줄
2. 번호 매긴 단계별 설명
3. 마무리 격려 한 줄 (예: "천천히 하시면 충분히 하실 수 있어요! 💪")
```

| **① 설정 변수 블록 분리** | 최상단의 변수만 변경하면 즉시 다른 전문 분야의 봇으로 변신 가능 |

| **③ 설계 구조의 범용성** | 효과가 검증된 뼈대(금지 목록, 우선순위 등)를 그대로 이식 가능 |


  

* #### **재사용 예시 (엑셀척척이)**

```
# ⚙️ 설정 변수 (재사용 시 이 블록만 수정하세요)

────────────────────────────
```


  `→ 나머지 규칙(친절·금지·우선순위)은 그대로 재활용`
